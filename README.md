# Shifting_Patriota
Aquí se desarrollara el segundo proyecto del ramo de Optimización NRC:7924 

Nombre del grupo: Los cautivos de til til

integrantes: Daniella Lecanda, Martias Alarcón, Elías Pérez.

Enunciado oficial: "Shifting de asignación de turnos basado en las restricciones en la curva de ventas multilocal y traslado de empleados entre locales, tomando en cuenta las restricciones que impone la ley chilena". 

En palabras menos tecnicas este proyecto aborda un problema de optimización operacional enfocado en la asignación eficiente de turnos de trabajo en múltiples sucursales, considerando la demanda variable representada por curvas de ventas, el traslado de empleados entre locales y las restricciones impuestas por la legislación laboral chilena.
